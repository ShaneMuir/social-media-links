Add social icons with links to profiles on your WordPress site.

## Description

The Social Links Plugin allows you to easily add and manage social media icons with links to your social media profiles. This plugin is perfect for bloggers and businesses who want to display social media links in an attractive and organized way.

### Features
- **Customizable Social Media Links**: Set your social media profile URLs.
- **Custom Icons**: Use your own icons for each social media link.
- **Adjustable Icon Size**: Choose the size of the icons to match your site's design.
- **Easy to Use Widget**: Add the social links widget to any widget area.

## Installation

1. Upload the `social-links` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Appearance > Widgets to add the Social Links widget to your desired widget area.

## Usage

1. Go to Appearance > Widgets.
2. Add the "Social Links Widget" to your desired widget area.
3. Configure the social media links and icons in the widget settings.
4. Save the widget settings to display the social media links on your site.

## Frequently Asked Questions

### How do I add the social media links to my site?
You can add the social media links by using the "Social Links Widget" in the Appearance > Widgets section of your WordPress admin.

### Can I use my own icons?
Yes, you can specify your own icon URLs in the widget settings for each social media link.

### Can I change the size of the icons?
Yes, you can set the icon size in the widget settings.

### Will this plugin slow down my site?
The Social Links plugin is lightweight and designed to minimally impact your site's performance.
